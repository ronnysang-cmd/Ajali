Ajali! is a community-driven emergency reporting platform designed to help deliver accident and emergency alerts to the appropriate authorities in real time. The system connects citizens and first responders because in emergencies every second matters.

## Problem Statement

Accidents and emergencies occur frequently in our country. The difference between life and death often depends on how quickly information reaches first responders.

Ajali! provides a localized digital solution that allows citizens to instantly report emergencies, share supporting evidence, and provide precise locations to ensure faster response and better coordination.

## Tech Stack

Frontend: React.js

Database: PostgreSQL
Maps Integration: Google Maps API
Wireframes: Figma (Mobile Friendly)
Testing: Jest (Frontend) and Minitests (Backend)

## MVP Features
## User Features

Users can create an account and log in

Users can create an incident report

Users can edit their incident reports

Users can delete their incident reports

Users can add geolocation (Latitude and Longitude) to reports

Users can update the geolocation attached to their reports

Users can upload images to support their claims

Users can upload videos to support their claims

The application displays a Google Map with markers showing incident locations

Admin Features

Admin can change the status of a report to:

Under Investigation

Rejected (false claim)

Resolved

Optional Features

Users receive real-time email notifications when the status of their report changes

Users receive real-time SMS notifications when the status of their report changes

Project Goal

To build a fast, reliable, and community-powered emergency alert system that improves response time and supports life-saving coordination between citizens and authorities.

Design Expectations

Mobile-friendly interface

Simple, clear, and fast user experience

Testing

Frontend testing using Jest

## Impact

Ajali! empowers citizens to actively contribute to emergency response while strengthening communication between the public and authorities.
